# pokemon-girl

## Prerequisite

1. Python 2.7
2. [pip](https://pip.pypa.io/en/stable/installing/)
3. [mkvirtualenv](http://virtualenvwrapper.readthedocs.io/en/latest/install.html#basic-installation)

## Quick Start

```
$ mkvirtualenv pokemon-girl
$ pip install -r requirements.txt
$ python cp.py
```
